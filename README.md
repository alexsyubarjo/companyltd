# alexsyubarjoltd — Landing Page

Landing page B2B untuk jasa pembuatan website profesional dan security pentesting/audit keamanan bisnis.

## Struktur

- `index.html` — 14 section lengkap (announcement bar, header + mega menu, hero, social proof, problem, solution, services, process, security audit, pricing, case studies, FAQ, CTA, footer)
- `styles.css` — desain modern navy/cyan, responsive desktop & mobile
- `script.js` — navigasi mobile, mega menu, smooth scroll, form konsultasi

## Menjalankan

Buka langsung di browser:

```bash
open website/index.html
```

Atau serve lokal:

```bash
cd website && python3 -m http.server 8080
# Buka http://localhost:8080
```
